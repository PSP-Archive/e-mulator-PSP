========================================================================
       WIN32 アプリケーション : win32
========================================================================


AppWizard はこの win32 アプリケーションを作成しました。  

このファイルには win32 アプリケーションを構成している各ファイルの概要
説明が含まれています。

win32.cpp
    このファイルは dll の中心となるソース ファイルです。

win32.dsp
    このファイル (プロジェクト ファイル) はプロジェクト レベルの情報を含み、
    シングル プロジェクトまたはサブ プロジェクトのビルドに使用されます。他の
    ユーザーとプロジェクト ファイル (.dsp) を共有できますが、メイク ファイル
    はローカルにエクスポートしてください。

/////////////////////////////////////////////////////////////////////////////
その他の標準ファイル:

StdAfx.h, StdAfx.cpp
    これらのファイルはプリコンパイル済みヘッダーファイル (PCH) win32.pch
    やプリコンパイル済み型ファイル StdAfx.obj を構築するために使用されます。
